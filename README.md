  - Database (mongoDB)
## Installing Dependencies
1. Backend
   - Go from root directory to backend directory
    ```
    cd backend/
    ```
   - Create virtualenv
   ```
   python -m venv env
   .\env\Scripts\activate
   pip install -r requirements.txt
   ```
   - Run the server:
    ```
    cd src/
    python manage.py runserver
    ```
2. frontend
   - Go from root directory to frontend directory
   ```
   cd frontend/
   npm i
   ```
   - Run the app
   ```
   npm start
   ```
